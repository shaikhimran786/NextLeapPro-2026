Improve the UI and UX of the Next Leap Pro website application.  
Next Leap Pro is a modern SaaS platform for students and professionals to Learn, Earn and Grow.  
Your task is to make the entire site more engaging, attractive and visually modern while keeping the layout clean and easy to use.

Apply the following improvements:

1. Modern Stylish Look and Feel
- Use a clean, spacious, modern SaaS design.
- Apply a consistent gradient theme that matches the brand colours:
  Magenta (#FF0099), Blue (#0066FF) and Green (#00CC66).
- Use smooth rounded cards, soft shadows, glass-morphism sections where suitable.
- Add beautiful hero illustrations or gradient shapes for depth.
- Improve typography hierarchy for better reading.

2. CTA (Call to Action) Enhancements
- Add more CTAs across the landing page and key screens.
- All CTAs should use gradient backgrounds using the magenta-blue-green combination.
- Add hover animations, glow effects, and subtle scale using CSS or Framer Motion.
- Examples:
  - “Join Now”
  - “Explore Events”
  - “Start Learning”
  - “Offer Your Services”
  - “Grow Your Network”
  - “Become a Mentor”

3. Improve Engagement Elements
- Add animated section transitions and scroll-based reveals.
- Add interactive cards for events, services and communities.
- Use micro-interactions on hover to make the app feel alive.
- Improve navbar design with gradient underline on active links.
- Add a sticky top navigation with smooth scroll.

4. Hero Section Update
- Add a bold hero headline:
  “Learn, Earn and Grow with Next Leap Pro”
- Add a sub-headline in simple language that explains the platform.
- Add a large primary CTA with gradient.
- Add a secondary CTA like “Browse Communities”.

5. Consistent UX Improvements
- Improve spacing and margins across the layout.
- Make buttons larger and friendly on mobile.
- Add card hover states, clickable areas and subtle shadows.
- Ensure light and fast loading images.
- Maintain accessibility (contrast, alt text, labels).

6. Apply these updates across:
- Landing page
- Events listing cards
- Communities listing
- Services marketplace
- Dashboard
- Login / register pages
- Pricing section

7. Output Requirements
- Update the code directly.
- Use Tailwind CSS classes for gradients, animations and spacing.
- Use Framer Motion where animations are needed.
- Keep components clean, reusable and well-structured.
- Do not break existing logic, routing or authentication.

Goal:
Make the Next Leap Pro interface feel premium, vibrant, modern and easy to use, powered by a lively magenta-blue-green gradient theme and strong CTAs that encourage students and professionals to take action.

Start applying these improvements directly in the code now.
