Build a fully functional SaaS web application called Next Leap Pro with the slogan Learn Earn and Grow. The platform will help students and working professionals learn new skills, earn through opportunities, and grow their career using events, communities, services, and mentoring.
This SaaS should function as a learning + earning + community ecosystem, similar in depth to commudle.com, but with additional monetisation and service-based earning features.

1. Tech Stack Requirements
Use the following:
* Next.js (latest)
* React.js + TypeScript
* Tailwind CSS
* Framer Motion for page transitions, scroll animations, micro interactions and hover effects
* Prisma ORM
* PostgreSQL
* NextAuth for authentication with:
    * Email/password
    * OAuth social logins (Google, GitHub, LinkedIn)
* Strong SEO support (meta tags, OG, structured data, clean routing, Core Web Vitals optimised)
* High-performance image optimisation
* Cached queries for fast data fetching
* Clean, scalable folder structure

2. Branding Requirements
* App Name: Next Leap Pro
* Slogan: Learn Earn and Grow
* Use provided logo (if not available, use placeholder)
* Colour theme: Magenta, Blue, Green gradient + white + light grey
* Use beautiful stock imagery and illustrations for landing page and key sections
* Modern SaaS design, lots of breathing space, professional but friendly
* Rounded cards, soft shadows, consistent spacing

3. User Roles & Access (Role-Based Platform)
Implement role-based access control (RBAC).
Supported roles:
* Student / Learner
* Working Professional
* Mentor
* Coach
* Speaker
* Organizer
* Sponsor
* Venue Partner
* Vendor
* Admin
A user can have multiple roles.
Each user profile includes: name, bio, avatar, skills, social links, role badges, experience, portfolio links.

4. Core Platform Capabilities (Learn – Earn – Grow)
Learn
* Learn through events, workshops, webinars, meetups, hackathons, conferences, college sessions, corporate training.
Earn
Users can earn money by offering services:
* Personal branding help
* Social media reels and video editing
* Graphic design (posters, banners, thumbnails)
* Website building
* Content writing & blogging
* Event management support
* Vendor management
* Speaking sessions
* Coaching & mentoring
* Corporate & college training workshops
Grow
* Build personal brand
* Grow followers through events
* Join communities
* Build network
* Showcase services & achievements

5. Multi-Currency Support
Make multi-currency support functionally available.
* Default currency = INR (Indian Rupees)
* Admin can enable more currencies (USD, EUR, GBP etc.)
* All prices stored in DB with both amount + currency code
* Currency selection dropdown (optional for user)
* Subscription, event tickets, services all support multi-currency pricing
* Mock payment flows but leave structure ready for Stripe later

6. Authentication System
Use NextAuth with:
* Email/password
* OAuth login: Google, GitHub, LinkedIn
* After login → onboarding role selection and interests selection
* All routes protected according to role & subscription

7. Subscription & Monetisation Model
Platform revenue comes from role-based subscription plans.
Create subscription tiers:
Free Plan
* Browse events
* Join free events
* Join communities
Pro Plan
* Students & Professionals
* Premium events
* Recordings
* Certificates
* Job board access
Creator / Organizer Plan
* Can create events
* Host paid sessions
* Analytics
* Community creation
* Branding support
Partner / Enterprise Plans
For sponsors, venue partners, vendors:
* Listing benefits
* Promotional placements
* Event partnerships
Subscription logic must be functional:
* Store plans in DB
* Store user subscription + expiry
* Restrict features according to plan
* Pricing page must reflect multi-currency
